# ZMK Configuration for Corne-ish Zen v2
The Corne (also known as crkbd or Helidox) is a completely open source split, compact keyboard. This ZMK configuration is for a wireless, modified, and closed-source build of the Corne known as the Corne-ish Zen v2.

## Use Case

The keybindings are configured for ergonomic usage, keeping high-frequency vim keys close to the thumb cluster.

## Specifications

- Corne 3x6 layout with epaper and RGB installed
- Key matrix identical to open-source Corne
